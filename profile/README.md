## Quick Start with ASAP Utilities

1. **Get ASAP Utilities** — Download the latest version from the official installer (one `.exe` file).  
2. **Install** — Run the installer with default settings. ASAP Utilities integrates directly into your Excel ribbon.  
3. **Launch Excel** — Open any workbook. You will see a new "ASAP Utilities" tab in the ribbon.  
4. **Start your first tool** — Click any tool, e.g., "Text" → "Change case" → "Proper case". Select a range and apply.  
5. **Explore the menu** — Browse through 300+ tools grouped logically: Text, Numbers, Dates, Select, Sheets, Formulas, etc.  
6. **Use shortcuts** — Many ASAP Utilities tools support keyboard shortcuts and quick-access customization.  
7. **Work normally** — ASAP Utilities becomes part of your daily Excel workflow without interrupting how you work.
